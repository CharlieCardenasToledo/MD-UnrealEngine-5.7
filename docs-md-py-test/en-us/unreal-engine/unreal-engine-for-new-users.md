# Unreal Engine for New Users

> Source: https://dev.epicgames.com/documentation/en-us/unreal-engine/unreal-engine-for-new-users

> Application Version: 5.7

In this tutorial series you will learn how to create a first person adventure game from scratch. You will start by installing and configuring Unreal Engine, learning about the interface and navigation controls, and selecting one or more learning tracks based on your interests. Finally, you will learn how to package your project so you can freely play it in any windows-based computer.

### Programmer Track

![Image](https://dev.epicgames.com/community/api/documentation/image/502131ed-0888-4b0c-b50b-d69210f343bf)

The [Programmer Track](https://dev.epicgames.com/documentation/en-us/unreal-engine/code-a-firstperson-adventure-game-in-unreal-engine) will teach you how to start coding a first person adventure game from scratch. You will learn how to setup your player character, add a camera, and control it inside a level. You will also learn how to create a pick up system where the player can pick up tools, weapons, or any other gameplay object.

### Designer Track

![Image](https://dev.epicgames.com/community/api/documentation/image/626a9288-31e5-4bba-a0ed-1c66d951f98a)

The [Designer Track](https://dev.epicgames.com/documentation/en-us/unreal-engine/design-a-puzzle-adventure-game-in-unreal-engine) will take you through the design process of building a level and several puzzles using blueprints to create a completely playable gameplay experience. You will learn how to create gameplay using Blueprints, as well as how to use and reuse gameplay elements to prototype a level using a process called grayboxing.

### Artist Track

![Image](https://dev.epicgames.com/community/api/documentation/image/b6e6695a-788f-4f2d-970e-58c07120f196)

The [Artist Track](https://dev.epicgames.com/documentation/en-us/unreal-engine/art-pass-for-a-puzzle-adventure-game) will walk you through the process of learning in-editor workflows and tools you can use to perform an art-pass on the worlds you build. You will learn beginner-friendly artist-driven workflows through practical game and gameplay-oriented examples. These can help you better understand how the various systems and tools of Unreal Engine work together to bring your games to life.

Now that you followed your desired learning track, you can package your project into a working game that you can run on any target machine.

## Last Step: Package your Project

![Image](https://dev.epicgames.com/community/api/documentation/image/9a851f85-cf91-4e40-9860-c18d36542101)

Read the [Packaging Unreal Engine Projects](https://dev.epicgames.com/documentation/en-us/unreal-engine/packaging-your-project) documentation to learn how to package your game to a target platform. Once packaged, you can run the game on any target platform independently of the Unreal Engine editor.
