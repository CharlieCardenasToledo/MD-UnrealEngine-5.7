# Parrot Game Sample

> Source: https://dev.epicgames.com/documentation/en-us/unreal-engine/parrot-game-sample-for-unreal-engine

> Application Version: 5.7

The Parrot Game Sample is a collaboration between Epic Games and veteran development studio [Secret Dimension](https://secretdimension.com/), created to support developers with Unity experience who are looking for a practical way to learn and develop a game in Unreal Engine.

Secret Dimension created the same game—a charming, pirate-themed 2.5D platformer with three levels, a title screen, and a full menu system—using the same assets in both Unreal Engine 5.6 and Unity 6. Their goal: achieve maximum parity between the two versions and capture every insight along the way.

![The menu UI of the Parrot Game Sample in Unreal Engine.](https://dev.epicgames.com/community/api/documentation/image/a1d95603-40dd-41ae-9f7e-a7604cc07122)

The result is the Parrot Game Sample, available for [free on Fab](https://www.fab.com/listings/844973a5-88a4-4870-a685-ab6d9770e203) in both Unity and Unreal Engine versions. Fully annotated and thoroughly documented, this project invites developers to dig into each project, read the documents and annotation, explore the tools, and absorb key lessons from Secret Dimension’s cross-engine journey.
