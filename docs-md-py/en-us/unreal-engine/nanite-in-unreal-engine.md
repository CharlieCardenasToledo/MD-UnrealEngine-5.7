# Nanite Virtualized Geometry

> Source: https://dev.epicgames.com/documentation/en-us/unreal-engine/nanite-in-unreal-engine

> Application Version: 5.7

Nanite is Unreal Engine's virtualized geometry system which uses an internal mesh format and rendering technology to render pixel scale detail and high object counts.
