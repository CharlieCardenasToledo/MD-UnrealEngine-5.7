# Module 5: Create the Game Over Screen

> Source: https://dev.epicgames.com/documentation/en-us/unreal-engine/module-5-create-the-game-over-screen

> Application Version: 5.7

Create the Game Over screen that displays when all the coins are collected using Blueprint Widgets and custom artwork.

[Video: V_HYkFVK](https://dev.epicgames.com/community/api/cms/videos/V_HYkFVK/embed.html)

In this fifth module, we show you how to use UMG to create the Game Over screen. You will reference the final artwork that is provided in the Platformer Project Files and learn to create custom Restart and Quit buttons.
