> Application Version: 5.7

### Description

VectorSquaredLength (v1)

Compute the Squared length of a vector : Length = |V||V|

Input(s) :
V - Vector to get squared length from

Output(s):
SquaredLength - Length of the input vector : SquaredLength = |V||V|
