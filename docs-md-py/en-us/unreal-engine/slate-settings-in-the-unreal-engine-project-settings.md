## Slate Settings

### Constraint Canvas

| **Section** | **Description** |
| --- | --- |
| **Explicit Canvas Child ZOrder** | Allow children of `SConstraintCanvas` to share render layers.  Children must set explicit ZOrder on their slots to control render order.  We recommend that you enable this setting for mobile platforms. |
