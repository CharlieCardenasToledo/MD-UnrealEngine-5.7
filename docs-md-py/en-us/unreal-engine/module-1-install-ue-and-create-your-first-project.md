# Module 1: Install UE and Create Your First Project

> Source: https://dev.epicgames.com/documentation/en-us/unreal-engine/module-1-install-ue-and-create-your-first-project

> Application Version: 5.7

Install Unreal Engine, create your project, and navigate the viewport to prepare for future development.

[Video: V_hRKYHb](https://dev.epicgames.com/community/api/cms/videos/V_hRKYHb/embed.html)

In this first module, we demonstrate how to install Unreal Engine using the Epic Games Launcher. Once installed, we walk you through creating your first project using the 3D platformer template and learning how to navigate the interface. You will also learn to create a level and save it to prepare for future development.
