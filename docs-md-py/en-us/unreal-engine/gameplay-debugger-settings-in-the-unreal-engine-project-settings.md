# Gameplay Debugger Settings

> Source: https://dev.epicgames.com/documentation/en-us/unreal-engine/gameplay-debugger-settings-in-the-unreal-engine-project-settings

> Application Version: 5.7

## Gameplay Debugger

| **Section** | **Description** |
| --- | --- |
| **Input** | The activation keys, shortcuts for categories and category rows. |
| **Display** | The settings for the debug canvas padding and the text shadow. |
| **Add-Ons** | Settings for game's add-ons: AI, BehaviorTree, EQs, NavMesh, Perception, NavGrid, GameHUD, and Spectator. |
