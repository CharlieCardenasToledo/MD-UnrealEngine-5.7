# Capture Manager

> Source: https://dev.epicgames.com/documentation/en-us/unreal-engine/capture-manager

> Application Version: 5.7

**Capture Manager** in [Live Link Hub](https://dev.epicgames.com/documentation/en-us/unreal-engine/live-link-hub-in-unreal-engine?application_version=5.5) is used to ingest footage captured on a variety of devices including audio, mono and stereo video, and mobile using the **Live Link Face** application. The footage is referred to as a take.
