# Material Functions Reference

> Source: https://dev.epicgames.com/documentation/en-us/unreal-engine/unreal-engine-material-functions-reference

> Application Version: 5.7

The reference pages below provide information and usage examples for all of the default **Material Functions** included with Unreal Engine. These are organized according to the categories in the [Material Editor](https://dev.epicgames.com/documentation/en-us/unreal-engine/unreal-engine-material-editor-ui) palette.
